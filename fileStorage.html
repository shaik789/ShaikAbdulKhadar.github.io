FILE STORAGE

A Secure File Storage System using Hybride Cryptography

Description

We are breathing in the air full of data. Now a days Internet is easily accessable to anyone. At many times, in order to have a safe backup, we store our data at places which may lack security where any anonymous intruders can access your data in just couple of attacks. With this project we are overcoming this drawback of storage locations by applying more than one encryption algorithm at each file.

Features and Working

We are using many encryption methods inlcuing Blowfish, DES and RSA. The uploaded files are passed through several cycles of encryption and a private key is generated. The key is a downloadable file with extension " .pem " which can be used to decode files again anytime you want.
